      <div class="step-num">2</div>

    </li>
    <li class="step-item">

    <li class="step-item">

      <div class="step-content">Click <strong>Link Issues</strong> and watch the live progress counter. Already-linked issues are skipped automatically.</div>
    </li>
  </ul>

  <h2>Why Bulk Issue Link Manager for Jira?</h2>
  <p>Jira Cloud's built-in link dialog links <strong>one issue at a time</strong>. If you need to mark a set of tickets as blocked by a dependency, link a sprint's worth of issues to an epic, or clean up hundreds of stale links, you're stuck repeating the same manual step over and over.</p>
  <p>Bulk Issue Link Manager for Jira was built to solve exactly that. It processes issues in smart batches, handles rate limiting silently, skips duplicates before they waste API calls, and gives you a clear colour-coded breakdown of what happened: successes in green, failures in amber with the exact reason grouped together.</p>
