# Features

Bulk Issue Link Manager for Jira adds three surfaces to Jira Cloud:

| Feature | Where it appears |
|---|---|
| [Link Issues in Bulk](./link-issues-in-bulk/) | `...` menu on any issue |
| [Unlink Issues in Bulk](./unlink-issues-in-bulk/) | `...` menu on any issue |
| [Link Summary Panel](./link-summary/) | Right sidebar on any issue |
